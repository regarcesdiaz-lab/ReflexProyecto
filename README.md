# Mi primera página web con Reflex

## 📌 Descripción del proyecto
Este proyecto consiste en el desarrollo de una página web básica utilizando el framework Reflex. La aplicación incluye un título principal, un mensaje de bienvenida y un botón interactivo que cambia el contenido al hacer clic. El objetivo es comprender la estructura básica de una aplicación web y el uso de herramientas modernas de desarrollo.

---

## ⚙️ Instrucciones de instalación

1. Instalar Python en la computadora (versión recomendada 3.11 o superior).
2. Verificar la instalación con el comando:
   python --version

3. Instalar Reflex ejecutando:
   pip install reflex

---

## ▶️ Cómo ejecutar la aplicación paso a paso

1. Clonar el repositorio:
   git clone https://github.com/TU_USUARIO/mi-web-reflex.git

2. Entrar a la carpeta del proyecto:
   cd mi-web-reflex

3. (Opcional en Windows, para evitar errores):
   set WATCHFILES_FORCE_POLLING=true
   set WATCHFILES_IGNORE_PERMISSION_DENIED=true

4. Ejecutar la aplicación:
   python -m reflex run

5. Abrir en el navegador:
   http://localhost:3000

---

## ✅ Funcionalidades

- Muestra un título principal.
- Presenta un texto de bienvenida.
- Incluye un botón interactivo que cambia el texto al hacer clic.
